=== Linux System shortcutkeys ===
..               : cd ..
...              : cd ../..
update           : update && upgrade -y
ll               : ls -alF  : vis alle filer, long list, med symboler
la               : ls -A    : vis alle filer, i kolonner, ikke systemdirs
l                : ls -CF   : viser ikke skjulte filer, i kolonner, med symboler

=== TMUX ===
  : shortcommands (outside)
  tm                               = tmux og/eller tmux a
  tml                              = tmux ls (lister alle kjørende)
  tmk -t x                         = tmux kill-session -t x (hvor x er indexnr)
  : shortcutkeys (inside)
  prefix + c                       = new window
  prefix + :                       = command mode
    new-session                    = starter ny session
    kill-session -t indexnr/navn   = lukker den spesifikke session
  alt + n                          = next panel
  alt + shift + n                  = neste vindu
  alt + ctrl + n                   = neste session
  : egne scripts
  ~/jrCuLiTe/tmux_status.sh : viser en meny, for å hente ut div. tmux statuser
  : viktige stier
  ~/jrCuLiTe/
  : viktige filer
  ~.tmux.conf
